# Анализ веб-сайта интернет-банка БСПБ

**Дата анализа:** Ноябрь 2025  
**Ссылка на файл:** [Website_analysis](./original_files/Website_Analysis.pdf)

## Шаг 1. Выбор веб-сайта

- Для анализа выбран интернет-банк БСПБ https://idemo.bspb.ru/
- Документация - [Руководство пользователя](https://cdn.bspb.ru/Rukovodstvo_polzovatelja_IB_67b34e84c0.pdf)



## Шаг 2. Основные функциональные элементы сайта

При осмотре и навигации стартовой страницы сайта, где отображена форма входа в личный кабинет интернет банка, были определенны основные функциональные элементы:

- гиперссылка «Физическое лицо»;
- гиперссылка «Юридическое лицо»;
- поле ввода логина;
- поле ввода пароля с кнопкой показа или скрытия просмотра;
- кнопка «Войти»;
- гиперссылка «Восстановить доступ»;
- гиперссылка «In English»;
- гиперссылка «Об Интернет-банке».

## Шаг 3. Пять сценариев использования на форме входа в личный кабинет

### Сценарий 1: Успешный вход юридического лица

**Цель:** Успешный доступ в личный кабинет зарегистрированного юридического лица с корректными учетными данными (multicustomer/secret).

**Контекст:** По документации, учитываю, что для входа в интернет-банк (помимо логина и пароля) и подтверждения операций, совершаемых в интернет-банке, используются одноразовые коды.

**Действия:**
1. На форме проверяю, что гиперссылка «Юридическое лицо» активна, выделена цветом (по умолчанию активна)
2. В поле логин ввожу – `multicustomer`
3. В поле пароль ввожу – `secret`
4. Нажимаю кнопку «Войти»
5. Система перенаправляет меня на форму ввода одноразового кода
6. В поле ввода кода ввожу `0000`, учитываю отведенное время для ввода
7. Нажимаю кнопку «Подтвердить»

**Результат:**
- Происходит перенаправление на главную страницу личного кабинета Интернет-банка
- Отображается стартовое модальное окно – «Вопросы, требующие вашего внимания»

---

### Сценарий 2: Переключение между типами пользователей

**Цель:** Успешное перенаправление с «Юридическое лицо» на «Физическое лицо» при нажатии на гиперссылку.

**Действия:**
1. На форме ввода по умолчанию гиперссылка «Юридическое лицо» активна
2. Нажимаю на гиперссылку «Физическое лицо»

**Результат:**
- Гиперссылка «Физическое лицо» визуально меняет цвет, значит она активна
- Появилась информация о входе в новый Интернет-банк для физических лиц
- Гиперссылка «Юридические лица» неактивна

---

### Сценарий 3: Проверка безопасности при неверном пароле

**Цель:** Проверка безопасности при входе с неверным паролем.

**Действия:**
1. В поле ввода логина ввожу – `multicustomer`
2. В поле ввода пароля вместо валидного пароля (`secret`) ввожу неверный – `secrets`
3. Нажимаю кнопку «Войти»

**Результат:**
- Перенаправление на форму ввода одноразового пароля не выполняется
- На форме входа под логотипом банка отображается сообщение – «Введены неверные данные»

---

### Сценарий 4: Просмотр пароля в открытом виде

**Цель:** Хочу просмотреть введенный пароль в открытом виде.

**Действия:**
1. В поле ввода пароля по умолчанию введен пароль – `secret`
2. Нажимаю на кнопку показа пароля (закрытый глаз)
3. Еще раз на эту же кнопку

**Результат:**
- После нажатия на кнопку показа пароля, символы пароля отображаются в открытом виде – `secret`
- Иконка кнопки поменяла вид на перечеркнутый закрытый глаз
- При повторном нажатии вместо символов появились точки
- Иконка кнопки поменяла вид на исходный (закрытый глаз)

---

### Сценарий 5: Восстановление доступа

**Цель:** Проверка восстановления доступа в личный кабинет интернет-банка.

**Действия:**
1. Нажимаю на гиперссылку «Восстановить доступ» расположенную на форме входа в личный кабинет

**Результат:**
- Происходит появление модального окна с заголовком «Забыли логин или пароль?»
- Отображается информация для восстановления доступа
